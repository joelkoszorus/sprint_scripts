# Scripting & Automation for Security Administration

Python automation scripts for system and security administration — AWS cloud lab, config management, and practical AI tool integration.

## Overview

This repo contains coursework from a scripting and automation course approached from a sysadmin and cybersecurity perspective. The focus is learning-by-doing: writing real Python scripts to solve problems a security admin would actually encounter on the job.

## Topics Covered

- Log analysis scripting
- Intrusion monitoring scripts
- Database interaction and public API integration
- Configuration automation
- Identity, password, and multi-factor authentication management
- AWS virtual server environment as a development platform
- AI-assisted problem solving and tool integration

## Course Outcomes

- Operate in a virtual server environment
- Apply scripting and debugging techniques to automate security administration tasks
- Use Git for source code control, change management, and code sharing
- Apply scripting to automate logging, monitoring, database interaction, and API use
- Develop systems for managing identity, passwords, and multi-factor authentication

## Stack

- **Language:** Python
- **Environment:** AWS (cloud instances)
- **Version Control:** Git
- **Focus:** Security administration, config management, automation

## Notes

Scripts in this repo are written to solve practical problems — not textbook examples. AI tools are incorporated intentionally, reflecting how they're applied in professional environments today.
